App Review Notes 템플릿(심사관용)

앱 핵심 목적

본 앱은 “프로젝트/태스크(매일 반복 포함)”를 관리하며, 잠금화면에서 오늘의 핵심 요약을 보여줍니다.

잠금화면 기능

잠금화면 위젯: 오늘 미완료/지연/반복 현황을 요약합니다.

Live Activity(선택): 핵심 1개 항목 또는 요약 카드를 표시합니다.

프라이버시

기본값은 프라이버시 모드(마스킹) ON이며, 제목 대신 요약/카운트 중심으로 표시됩니다.

단축어(Shortcuts) 사용(선택)

사용자는 App Intent “RefreshLiveActivity”를 단축어에 추가하여, Live Activity를 주기적으로 갱신할 수 있습니다(예: 8시간 간격).

단축어는 선택 기능이며, 위젯만으로도 핵심 기능이 동작합니다.

테스트 방법(심사관 재현 단계)

(1) 앱 실행 → 프로젝트 생성 → 태스크 2개 추가(일반 1, 매일 반복 1)

(2) 잠금화면에 위젯 추가 → 요약 표시 확인

(3) 앱에서 태스크 완료 체크 → 위젯 카운트 변화 확인(즉시 반영은 OS 정책상 지연될 수 있음)

(4) Live Activity 활성화(앱 내 버튼 또는 단축어) → 표시 확인

(5) 설정에서 프라이버시 모드 토글 → 잠금화면 노출 변화 확인
