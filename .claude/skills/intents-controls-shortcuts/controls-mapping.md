Controls 매핑 표(Phase 1)

원칙

Controls는 최대 3개 버튼.

리스트/탐색 UI 금지. “명령 버튼”만.

권장 매핑

CompleteNextTask

버튼 라벨: “완료”

파라미터: scope=pinned, allowRecurring=true

폴백: pinned 없으면 today

QuickAddTask

버튼 라벨: “추가”

파라미터: type=oneOff, projectId=nil(기본 정책 적용)

입력 텍스트는 최소 형태(가능 범위)로 처리

RefreshLiveActivity

버튼 라벨: “갱신”

파라미터: displayMode=pinnedSummary

iOS 17 폴백

Controls 없음 → 위젯 탭으로 앱을 열어 동일 기능 수행
