단축어 자동화(8시간 Live Activity 갱신) 설정 안내(사용자용 초안)

목표

8시간마다 RefreshLiveActivity를 실행하여 Live Activity 표시를 갱신합니다.

준비

앱에서 Live Activity 권한이 허용되어 있어야 합니다.

프라이버시 모드는 기본으로 “마스킹 ON”입니다(잠금화면 노출 최소화).

설정 절차(개요)

단축어 앱 열기 → “자동화” 탭

“개인용 자동화 생성”

트리거 선택

시간 기준(예: 매일 특정 시각) 또는 “시간 간격”(가능한 형태로 선택)

동작 추가 → 앱의 “RefreshLiveActivity” 선택

displayMode 선택

권장: pinnedSummary (핀 없으면 todaySummary로 자동 폴백)

“실행 전에 묻기” 옵션 정책

가능한 경우 “실행 전에 묻기”를 꺼서 자동 실행되게 설정

저장 후 수동 실행으로 1회 테스트

문제 해결

Live Activity가 표시되지 않으면

Live Activity 권한이 꺼져 있는지 확인

잠금화면 위젯은 별도로 추가해야 합니다(위젯은 최소 기능 폴백)

자동화가 불규칙하게 실행되면

단축어/권한 설정을 재확인하고, 실패해도 다음 주기에 자동 회복하도록 설계되어 있습니다.
