길이 예산(권장, Phase 1)

원칙

잠금화면은 "한눈에"가 목표이므로 짧게.

길이 제한은 실제 UI에서 달라질 수 있으나, 문구 작성 시 아래를 기본으로 한다.

권장 문자 수(한국어 기준, 공백 포함)

Widget Inline: 18자 내

Widget Rectangular 1라인(Top 항목): 18~24자 내(말줄임 허용)

Live Activity 핵심 1라인: 20~28자 내(말줄임 허용)

인텐트 반환 메시지: 18~30자 내(가능하면 더 짧게)

규칙

숫자 표기는 "미완료 3 · 지연 1"처럼 구분점(·) 사용

가능한 한 조사/서술어 최소화

"앱을 열어 확인" 같은 안내는 오류/폴백에서만 사용
