## 스크린샷 사양

| 기기 유형 | 최소 해상도 | 권장 해상도 |
|----------|------------|------------|
| 휴대전화 | 320 x 569 px | 1080 x 1920 px (FHD) |

## 스크린샷 7장 스토리보드

### 1️⃣ 프로젝트/할 일 생성
**파일명**: `01_create_project_task.png`

**화면 구성**:
- 프로젝트 생성 화면 또는 태스크 추가 화면
- 시작일/마감일 선택 UI 표시
- 반복(Daily) 옵션 표시

**캡션 (KR)**:
```
프로젝트와 할 일을 손쉽게 생성
시작일, 마감일, 반복 설정까지
```

**캡션 (EN)**:
```
Create projects and tasks easily
Set start date, due date, and recurring options
```

**촬영 가이드**:
1. 앱 실행 → + 버튼 탭
2. "새 프로젝트" 또는 "새 태스크" 화면
3. 날짜 선택 다이얼로그가 보이면 좋음
4. "반복" 토글이 보이도록

---

### 2️⃣ 홈 화면 위젯 요약
**파일명**: `02_home_widget.png`

**화면 구성**:
- 홈 화면에 LivePlan 위젯 배치
- 위젯에 "미완료 3 · 지연 1" 등 카운트 표시
- 배경은 일반적인 홈 화면 (기본 배경화면)

**캡션 (KR)**:
```
홈 화면에서 바로 확인
미완료, 지연, 임박 항목을 한눈에
```

**캡션 (EN)**:
```
Check right from home screen
See outstanding, overdue, due soon at a glance
```

**촬영 가이드**:
1. 홈 화면에 위젯 추가 (2x2 또는 4x2)
2. 샘플 데이터: 프로젝트 1개, 태스크 5개 (2개 완료, 1개 지연)
3. 밝은 배경화면 사용 권장
4. 위젯이 화면 중앙~상단에 위치

---

### 3️⃣ 반복 항목 개념
**파일명**: `03_recurring_concept.png`

**화면 구성**:
- 태스크 목록 화면
- 반복 태스크에 반복 아이콘/배지 표시
- "오늘 완료 → 내일 리셋" 개념 시각화
  - 옵션 A: 반복 태스크 상세 화면
  - 옵션 B: 태스크 목록에서 반복 아이콘 강조

**캡션 (KR)**:
```
매일 반복 태스크
오늘 완료하면 내일 다시 표시
습관 형성에 최적
```

**캡션 (EN)**:
```
Daily recurring tasks
Complete today, returns tomorrow
Perfect for building habits
```

**촬영 가이드**:
1. 반복 태스크 2-3개 생성
2. 반복 아이콘(🔄 또는 ↻)이 명확히 보이도록
3. 가능하면 "매일 반복" 텍스트가 보이는 상세 화면

---

### 4️⃣ 프라이버시 모드
**파일명**: `04_privacy_mode.png`

**화면 구성**:
- 프라이버시 모드 ON 상태의 위젯 또는 목록
- 태스크 제목이 마스킹된 상태 ("••••••" 또는 "할 일 1")
- 설정 화면에서 프라이버시 토글 (옵션)

**캡션 (KR)**:
```
프라이버시 모드
홈 화면에서 제목 숨김
주변 시선으로부터 안전하게
```

**캡션 (EN)**:
```
Privacy mode
Hide titles on home screen
Keep your tasks private
```

**촬영 가이드**:
1. 설정 → 프라이버시 모드 ON
2. 위젯에서 마스킹된 제목 확인
3. 또는 앱 내 목록에서 마스킹 표시
4. "숫자만 보이는" 상태가 이상적

---

### 5️⃣ Board 뷰 (보드 뷰)
**파일명**: `05_board_view.png`

**화면 구성**:
- 보드 뷰 (칸반 스타일) 화면
- 컬럼: To Do / Doing / Done
- 태스크 카드가 컬럼에 배치된 모습

**캡션 (KR)**:
```
보드 뷰로 작업 흐름 관리
드래그 앤 드롭으로 상태 변경
```

**캡션 (EN)**:
```
Manage workflow with Board view
Drag and drop to change status
```

**촬영 가이드**:
1. 프로젝트 상세 → 뷰 전환 (보드)
2. 각 컬럼에 태스크 2-3개씩 배치
3. 수평 스크롤 레이아웃
4. 카드가 드래그 중인 상태 (옵션)

---

### 6️⃣ Quick Settings Tile
**파일명**: `06_quick_settings_tile.png`

**화면 구성**:
- 빠른 설정 패널 (상단에서 스와이프)
- LivePlan 타일 표시 ("다음 할 일 완료" 또는 앱 아이콘)
- 타일이 하이라이트된 상태

**캡션 (KR)**:
```
빠른 설정에서 바로 완료
앱을 열지 않고 체크
```

**캡션 (EN)**:
```
Complete from Quick Settings
Check off without opening app
```

**촬영 가이드**:
1. 빠른 설정 패널 편집 → LivePlan 타일 추가
2. 패널을 펼쳐서 타일이 보이도록
3. 타일 터치 직전 또는 직후 상태
4. 화면 녹화 후 캡처도 가능

---

### 7️⃣ App Shortcuts
**파일명**: `07_app_shortcuts.png`

**화면 구성**:
- 앱 아이콘을 길게 눌렀을 때 나오는 바로가기 메뉴
- "빠른 추가", "오늘 보기" 등 옵션 표시
- 앱 아이콘과 바로가기 팝업이 함께 보이도록

**캡션 (KR)**:
```
앱 바로가기로 빠른 접근
길게 눌러서 바로 태스크 추가
```

**캡션 (EN)**:
```
Quick access with App Shortcuts
Long-press to add task instantly
```

**촬영 가이드**:
1. 홈 화면에서 앱 아이콘 길게 누름
2. 바로가기 메뉴가 펼쳐진 상태
3. "빠른 추가" 옵션이 명확히 보이도록
4. 손가락이 화면에 없는 상태로 캡처

## 촬영 팁

### 데모 데이터 준비
```
프로젝트: "업무", "개인"
태스크 예시:
- 회의 준비 (일반)
- 운동하기 (반복)
- 영어 공부 (반복)
- 프레젠테이션 마감 (마감일: 내일)
- 보고서 작성 (지연)
```

### 기기 설정
- 시간: 오전 10:00 정도 (자연스러운 시간)
- 배터리: 80% 이상
- 알림 표시줄: 깨끗하게 정리
- 다크모드: 라이트 모드 권장 (또는 둘 다)

### 스크린샷 앱
- Android Studio → Device Mirroring → Screenshot
- adb shell screencap
- 기기 내장 스크린샷 기능

### 후처리 (옵션)
- 상태 표시줄 정리 (시간/배터리 통일)
- 디바이스 프레임 추가 (Figma/Canva)
- 캡션 오버레이 추가
