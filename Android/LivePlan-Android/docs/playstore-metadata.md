## 상세 설명 (Full Description, 4000자 이내)

### 한국어 (KR)

```
LivePlan은 홈 화면에서 오늘의 할 일을 바로 확인하고 관리할 수 있는 할 일 관리 앱입니다.

🎯 핵심 기능

• 프로젝트별 할 일 관리
  - 프로젝트 단위로 태스크를 정리하세요
  - 시작일/마감일 설정으로 일정 관리
  - 대표 프로젝트를 지정해 중요한 일에 집중

• 매일 반복 기능
  - 매일 해야 하는 일을 반복 태스크로 설정
  - 오늘 완료하면 내일 자동으로 다시 표시
  - 습관 형성과 루틴 관리에 최적

• 홈 화면 위젯
  - 앱을 열지 않고 홈 화면에서 바로 확인
  - 미완료/지연/임박 항목을 한눈에
  - 남은 할 일 개수 카운터 표시

• Quick Settings 타일
  - 빠른 설정 패널에서 바로 완료 처리
  - "다음 할 일 완료" 버튼으로 빠른 체크

• 앱 바로가기 (App Shortcuts)
  - 홈 화면에서 길게 눌러 빠른 태스크 추가
  - 자주 쓰는 기능에 빠르게 접근

🔒 프라이버시 우선 설계

• 홈 화면은 주변 사람이 볼 수 있습니다
• 기본값은 프라이버시 모드(마스킹) ON
• 할 일 제목을 숨기고 개수만 표시하는 옵션
• 완전히 로컬 저장, 서버 전송 없음

📱 Android 전용 기능

• Material You 디자인 적용
• 다이나믹 컬러 지원 (Android 12+)
• 홈 화면 위젯 (Glance)
• Quick Settings 타일
• 앱 바로가기 (App Shortcuts)

💡 이런 분께 추천합니다

• 매일 해야 할 일을 놓치지 않고 관리하고 싶은 분
• 앱을 열지 않고 홈 화면에서 바로 확인하고 싶은 분
• 프라이버시를 중요하게 생각하는 분
• 단순하고 가벼운 할 일 앱을 찾는 분

⚡ 가볍고 빠른 앱

• 외부 SDK 최소화, 앱 용량 최적화
• 배터리 소모 최소화 설계
• 오프라인 완벽 지원

LivePlan으로 오늘의 할 일을 더 쉽게 관리하세요.
```

LivePlan is a task management app that lets you view and manage today's tasks right from your home screen.

  - Organize tasks by project

  - Perfect for habit building and routine management

  - Check tasks without opening the app
  - See outstanding, overdue, and due soon items at a glance

  - Complete tasks directly from Quick Settings panel

• App Shortcuts
  - Long-press to quickly add new tasks
  - Fast access to frequently used features

• Home screens can be seen by others nearby

• Material You design

• Quick Settings Tile
• App Shortcuts

• Those who want to check tasks from home screen without opening apps
• Those who value privacy
• Those looking for a simple and lightweight task app

⚡ Light and Fast

• Minimal external SDKs, optimized app size
• Designed for minimal battery consumption
• Full offline support

Manage your daily tasks more easily with LivePlan.

## 심사관 메모 (Developer Notes)

```
LivePlan is a lock-screen/home-screen-first task management app.

Core Features:
1. Widget: Shows task summary (top 3 + count)
2. Quick Settings Tile: "Complete Next Task" action
3. App Shortcuts: Quick add task

Privacy Default:
- Privacy mode is ON by default
- Task titles are masked on widget when privacy mode is enabled
- All data is stored locally only

Testing Steps:
1. Create a new project (tap + button)
2. Add 3+ tasks (mix of one-off and daily recurring)
3. Add widget to home screen
4. Verify widget shows task count
5. Complete a task and verify widget updates
6. Toggle privacy mode in Settings and verify masking

Optional Features:
- Quick Settings Tile requires adding from tile editor
- App Shortcuts available via long-press on app icon

No account/login required.
No network permissions used.
No analytics SDK included.
```
