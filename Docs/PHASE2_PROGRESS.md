1. 티켓 완료 시:
   - 상태를 `⬜` → `✅`로 변경
   - 완료일 입력 (YYYY-MM-DD)

2. 마일스톤 업데이트:
   - 진행률 바 업데이트 (`░` → `█`)
   - 완료 수 업데이트

3. 전체 진행률 업데이트:
   - 상단 요약 테이블 갱신
