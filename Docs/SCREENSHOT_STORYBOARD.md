## 7장 스토리보드

### 스크린샷 1: 프로젝트/할 일 생성

**화면**: 프로젝트 목록 + 태스크 목록

**구성 요소**:
- 2-3개 프로젝트 표시 (예: "업무", "개인", "건강")
- 각 프로젝트에 미완료 카운트 표시
- 하나의 프로젝트 펼쳐서 태스크 3-5개 보이기
- 다양한 우선순위 (P1, P2, P4) 표시

**캡션 (KR)**: "프로젝트로 체계적으로 관리하세요"
**캡션 (EN)**: "Organize with Projects"

**포인트**:
- 깔끔한 UI, 우선순위 색상 강조
- 반복 태스크 아이콘 포함

---

### 스크린샷 2: 잠금화면 위젯 요약

**화면**: 잠금화면 (iPhone Lock Screen) + LivePlan 위젯

**구성 요소**:
- 실제 잠금화면 배경 (심플한 배경 권장)
- LivePlan 직사각형 위젯 (Top 3 + 카운트)
- 시계 아래 위젯 영역에 배치
- "미완료 5 · 지연 1" 형태의 카운터

**캡션 (KR)**: "잠금화면에서 할 일을 한눈에"
**캡션 (EN)**: "Check Tasks Without Unlocking"

**포인트**:
- 잠금화면 핵심 가치 강조
- 마스킹 모드로 프라이버시 보호 표현 가능

---

### 스크린샷 3: 보드 뷰 (Board View)

**화면**: 프로젝트 상세 - 보드 뷰

**구성 요소**:
- To Do / Doing / Done 3개 컬럼
- 각 컬럼에 2-3개 태스크 카드
- 우선순위/태그 표시
- "Doing" 컬럼에 진행 중 태스크 강조

**캡션 (KR)**: "보드에서 진행 상황을 파악하세요"
**캡션 (EN)**: "Track Progress with Board View"

**포인트**:
- Asana/Trello 스타일 친숙한 UI
- 드래그 힌트 (손가락 아이콘 오버레이 선택)

---

### 스크린샷 4: 캘린더 뷰 (Calendar View)

**화면**: 프로젝트 상세 - 캘린더 뷰

**구성 요소**:
- 월간 캘린더 그리드
- 날짜별 태스크 도트 표시
- 특정 날짜 선택 시 태스크 목록 표시
- 오늘 날짜 강조

**캡션 (KR)**: "마감일 기준으로 일정 관리"
**캡션 (EN)**: "Schedule by Due Date"

**포인트**:
- 깔끔한 캘린더 UI
- overdue/dueSoon 색상 구분

---

### 스크린샷 5: 프라이버시 모드

**화면**: 설정 > 프라이버시 + 잠금화면 비교

**구성 요소**:
- 설정 화면: 프라이버시 레벨 선택 UI
- 잠금화면 위젯: 마스킹된 "할 일 1, 할 일 2"
- Before/After 스플릿 (선택)

**캡션 (KR)**: "잠금화면에서 민감한 정보를 보호하세요"
**캡션 (EN)**: "Protect Sensitive Info on Lock Screen"

**포인트**:
- 프라이버시 기본값 Level 1 강조
- "주변 사람이 볼 수 없도록" 메시지

---

### 스크린샷 6: Live Activity

**화면**: Dynamic Island 또는 잠금화면 Live Activity

**구성 요소**:
- Dynamic Island 확장 상태 (또는 잠금화면 Live Activity)
- "요약 + 1개 핵심" 표시
- 우선순위/상태 아이콘

**캡션 (KR)**: "실시간으로 오늘 할 일을 확인 (선택 기능)"
**캡션 (EN)**: "Live Activity for Today's Tasks (Optional)"

**포인트**:
- "선택 기능" 명시
- 8시간 제한 설명은 캡션에서 생략 (App Review Notes에서 설명)

---

### 스크린샷 7: Controls (iOS 18+)

**화면**: 잠금화면 또는 Control Center의 LivePlan Controls

**구성 요소**:
- Control Center에 배치된 LivePlan 버튼들
- Complete / Add / Start 버튼 표시
- 실행 결과 메시지 (선택)

**캡션 (KR)**: "잠금화면에서 바로 완료/추가 (iOS 18+, 선택)"
**캡션 (EN)**: "Complete & Add from Lock Screen (iOS 18+, Optional)"

**포인트**:
- iOS 18+ 전용임을 명시
- "선택 기능" 표현
