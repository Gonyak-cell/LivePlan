## 상세 설명 (4000자 이내)

### 잠금화면에서 시작하는 하루

LivePlan은 잠금화면에서 오늘 해야 할 일을 바로 확인할 수 있는 할 일 관리 앱입니다.
복잡한 앱을 열지 않아도, 잠금화면 위젯 하나로 오늘의 핵심 태스크를 빠르게 파악하세요.

### 주요 기능

**프로젝트 & 태스크 관리**
- 프로젝트별로 태스크를 구분하고 관리
- 섹션으로 태스크 그룹화
- 태그로 유연한 분류
- 우선순위(P1~P4) 설정으로 중요한 일 먼저

**뷰 전환 (리스트/보드/캘린더)**
- 리스트 뷰: 전통적인 체크리스트 형태
- 보드 뷰: 할 일/진행 중/완료 상태 한눈에
- 캘린더 뷰: 마감일 기준 월간 일정 확인

**매일 반복 태스크**
- 매일/주간/월간 반복 설정
- 오늘 완료해도 내일 자동으로 다시 표시 (습관 관리용)
- 미완료 누적 방식도 지원 (업무 관리용)

**잠금화면 위젯**
- 미완료 태스크 요약 (Top 3 + 남은 개수)
- 지연/임박/반복 카운트 표시
- 대표 프로젝트 또는 오늘 요약 선택

**프라이버시 모드 (기본 활성화)**
- 잠금화면은 주변 사람이 볼 수 있습니다
- 기본값은 제목 마스킹 (카운트 중심 표시)
- 원하면 원문 표시로 전환 가능

**필터 & 검색**
- 오늘/임박/지연/우선순위별 빌트인 필터
- 사용자 정의 필터 저장
- 프로젝트/태스크/태그 로컬 검색

### 선택 기능 (필수 아님)

**Live Activity (iOS 17+)**
- 다이나믹 아일랜드에 핵심 태스크 1개 표시
- 8시간 활성 제한 (시스템 정책)
- 단축어 자동화로 갱신 가능 (선택)

**단축어 연동**
- RefreshLiveActivity: Live Activity 수동 갱신
- CompleteNextTask: 최우선 태스크 완료
- QuickAddTask: 빠른 태스크 추가
- StartNextTask: 태스크 시작 (진행 중 전환)

**Controls (iOS 18+)**
- 잠금화면/제어 센터에서 완료/추가 버튼
- Live Activity 없이도 핵심 기능 사용 가능

### 위젯은 즉시 갱신되지 않습니다

iOS 정책상 위젯은 최소 5분 간격으로 갱신됩니다.
앱에서 완료해도 위젯에 바로 반영되지 않을 수 있습니다.
이 점은 모든 위젯 앱에 동일하게 적용됩니다.
